Project Info Tab


Overview

New quotes are initiated in this tab.


Customer Info

Set the Customer for the current quote from the dropdown. A Customer must be selected in order to save or print quotes from the SUM+IT app. You can also add or edit Customers here.

  • Add Customer: With no Customer selected (top selection), click "Add/Edit Customer". Fill out the Customer Information fields in the form box below. Click the "Save" button to save, or "Cancel" to exit.
  • Edit Customer: If you need to edit information from a pre-existing Customer, select the Customer from the dropdown list. Click the "Add/Edit Customer" button in the top bar. Make the necessary changes and click the "Save" button below.


Project Details

Set the Quote Number, Job Tag and Address/Notes for the current quote.

  • Quote Number: Enter a quote number to identify your quote. A Quote Number is required in order to save or print quotes from the SUM+IT app. Make sure that the number you choose is unique from other quotes. If you select a number that has been used before, when you attempt to save or print the quote, you will be prompted so that you can confirm if you would like to overwrite the previous quote with the same Quote Number.
  • Job Tag: Enter the Job Tag name. The Job Tag field is not required. The Job Tag will appear on printed quotes.
  • Address/Notes: Enter the job site address. The Address/Notes field is not a required input. This information will appear on printed quotes.
  • Estimator: If your company has multiple users, quotes can be compiled together so that all the quotes for the company will be displayed in the "View Quotes" tab. You can enter the various Estimator names in the "Company Info" tab.


Loading

Set the Loading fields for the current quote. The "Default Settings" button will match all the fields in the Loading section with the default values from the "Job Defaults" tab. The fields in the loading panel will display differently depending on whether you are using the USA or Canada version.

USA version

  • Category: For the current version, Category is limited to Residential and to clear spans less than 40 feet.
  • Ground Snow Load: Select the Snow Load appropriate for the geographical region of the current quote. Top Chord Live load will display in green lettering to the above right of the input box.
  • Top and Bottom Chord Dead Load Inputs: Enter the dead loads to be used for the specific project.
  • Additional Dropdowns: Terrain Category, Exposure, Thermal Factor and Risk Category follow ASCE guidelines to determine the Top Chord Live Load.

Canada version

  • Category: Select either Residential or Commercial for the current quote. For the current version, do not use for pricing Commercial jobs with a clear span greater than 40 feet.
  • Snow Load: Select the Snow Load appropriate for the geographical region of the current quote. Use the full Ground Snow Load value without factoring down.
  • Rain Load: Select the Rain Load appropriate for the geographical region of the current quote.

USA and Canada version

  • Floor Load: This is specific for Floor Trusses. You can select the Floor Loading for the job. If you have certain areas with diffent loads this can be changed later on in the "Floor Estimator". Whatever you select here will become the default load setting for the job.
  • Deflection Limits: This is also specific for Floor Trusses. You can change the deflection limits to allow an option for a stiffer floor. If L/480 is selected, maximum spans on floor trusses will be reduced accordingly.
  • Bearing Width: Select the wall bearing width for the current quote. Bearing width is important for determining Girder Plies in cases where end reactions exceed bearing size limitations. Compression limitations are based on SPF values perpendicular to grain.
  • Overhang: Select the horizontal Overhang distance in inches.


Reset Quote Button

If you have finished a quote and want to start another one, click the "Reset Quote" button. The "Reset Quote" button will clear all stored information in "Project Info", "Floor Estimator", "Truss Estimator", and "Print Quote" to either be empty or matching the Job Defaults settings to prepare the app for a new quote.



Floor Estimator


Overview

The Floor Estimator takes information set by the user in "Product Codes" and makes them available for instant EWP floor quotes. You can also price floor trusses in 2x3 and 2x4 width from 12" to 24" deep.


Joist Section 1, 2, etc

If you have more than one joist type or spacing in the floor you are quoting, you can break up the floor into sections. The "+" button in the top right corner of the Joist Section panel will add as many areas as you need to complete the quote.

  • Floor Area: Enter the floor area in square feet for this section of floor that you want to price.
  • Spacing Enter the floor joist spacing for this section of floor.
  • Joist Type: Select the joist series from the dropdown. You can use your own inventory of joists by entering the joist products you sell along with your pricing in the "Product Codes" tab under the "Joists" heading.
  • Rim Board Length: Rim Boards must be entered separately. Total up the linear feet of Rim Board required and enter it here.
  • Rim Board Type: Select the Rim Board from the dropdown to match the joist type. You can use your own inventory of Rim Board by entering the rim products you sell along with your pricing in the "Product Codes" tab under the "Rimboards" heading.


Floor Trusses Section 1, 2, etc

If you have more than one floor truss in the floor you are quoting, you can break up the floor into sections. The "+" button in the top right corner of the Floor Truss section panel will add as many areas as you need to complete the quote.

  • Span: Enter the span (width) in decimal feet for this section of floor trusses.
  • Run: Enter the joist run (length) in decimal feet for this section of floor trusses.
  • Spacing: Select the joist spacing. This and other selections can be changed at any time. If you set spacing to 19.2" but after completing all inputs it is apparent that the inputs exceed limitations, you can try reducing the spacing to 12" or 16" to try to remedy the problem.
  • Depth: Select the depth of the floor.
  • Width: Select between 2x3 and 2x4 width. 2x3 Floor Trusses are limited to 20 feet maximum span. Beyond 20 feet, use 2x4 width.
  • End 1 / End 2: These dropdowns allow users to select from six floor truss end options - Vertical, Double Vertical, Ribbon Strip, Top Chord Bearing, Mid Block Bearing, or Trimmable. If you use trimmable inserts, or if you use a 2x6 or 2x8 trimmable end block, you can use this option by putting a cost for the material in the "Cost Variables" tab in the Material Cost panel.
  • Start / Finish with End Wall Ladder (checkboxes): If the section being quoted will start or end with an End Wall Ladder, you can check the appropriate boxes here. Be sure that you have set up your company defaults in the "Job Defaults" tab. When setting your defaults, look for End Wall Ladder Stud Spacing to set the stud spacing and End Wall Ladder Width to set whether you supply end wall ladders built with 2x3 or 2x4 lumber.
  • Chase Opening: If the section you are working on has a chase opening, you can use this checkbox. Keep in mind that chase opening widths may need to be reduced as floor trusses near their span limits.
  • Web Pattern: Select between K Web and Warren truss web configuration. If working with L/480 increased deflection limitations, try using Warren for slightly improved maximum spans.
  • Floor Load: The Floor Load that was set in Project Info will be the default load for each section of floor trusses you add into the quote. If you have an anomoly in your job, you can switch between loads for the specific section.


Beams

Additional products can be added to the floor quote. In this section, multiple selections can be added together.

  • Product: Select the beam or item to add to the quote from the dropdown. This dropdown takes the products that are stored in "Product Codes" under the "Beams", "Rimboards", and "Misc. Items" headings.
  • Length: Enter the total length in feet and click the add "+" button below. Repeat the process to add as many additional items as necessary. If you need to edit the amount, select the same product from the dropdown, enter the proper length and click the add "+" button to overwrite.


Hangers

  • Product: Hangers added to the "Product Codes" tab under the "Hangers" heading will appear in the "Product" dropdown. Select a hanger from the dropdown.
  • Quantity: Enter the quantity for the selected hanger and add to the list below.



Truss Estimator


Overview

The "Truss Estimator" uses the parameters set in "Cost Variables" and the design preferences set in "Job Defaults" to produce accurate roof truss quotes with ease and speed.


Section 1, 2, etc.

When quoting, a plan can be broken down into multiple sections. Tackle large plans a section at a time. When a section is completed, click the "+" button in the top right corner of the "Section 1" section, and a duplicate "Section 2" will appear below. You must have a span and run entered before you can add another Section.


Top Row Inputs

The five top row inputs of the "Truss Estimator" are required. There are two options for the entering method, Span and Run, which is the default method, or Area Input.


Default Input Method

Divide the plan you are quoting into rectangular sections with a specific span and truss run. Each section will start and end with a truss.

  • Span: Enter the truss span or width for the specific section. If the ends are cantilievered, include this in the span.
  • Truss Run: Enter the truss run or length of the section.
  • Slope: Enter the nearest slope to match the plan being quoted. If you have an unequal sloped hip roof, select the slope of the full span hip trusses.
  • Truss Spacing: Select the desired truss spacing from the dropdown.
  • Heel Condition: Select from the dropdown box the heel condition for the specific section that best matches the plan. Approximate heels for 2x4 chords: 1/4" butt cut = 4", Wedge = 6", Slider = 8-10", End Vertical = 12+".
  • Canted Ends: If the eave detail includes cantilevered ends, click the "Canted Ends" checkbox below the "Heel Condition" dropdown. Be sure to include the full truss span including the cantilever for the Area calculation.

Area Input Method

When a section cannot be easily squared off into a span and run input (example: angled walls or slight jogs), you can toggle Area Input. Top row inputs change to Area and Span. Take the average span of the section for the Span input if all spans are not the same. Each section will start and end with a truss.

  • Area: Enter the area in square feet for the specific section by calculating the truss span times the length (or run). If you are quoting the plan in a single area, enter the total square foot coverage of the roof truss system.
  • Span: Enter the truss span in decimal feet for the specific section. If you are covering an area with multiple spans, enter the most common or average span. Currently, the maximum clear span is 40 feet. You may enter longer spans, but intermediate bearings will be assumed.
  • Slope: Enter the nearest slope to match the plan being quoted. If you have an unequal sloped hip roof, select the slope of the full span hip trusses.
  • Truss Spacing: Select the desired truss spacing from the dropdown.
  • Heel Condition: Select from the dropdown box the heel condition for the specific section that best matches the plan. Approximate heels for 2x4 chords: 1/4" butt cut = 4", Wedge = 6", Slider = 8-10", End Vertical = 12+".
  • Canted Ends: If the eave detail includes cantilevered ends, click the "Canted Ends" checkbox below the "Heel Condition" dropdown. Be sure to include the full truss span including the cantilever for the Area calculation.

Truss Height, Area and Truss Count Display

Directly below the five Top Row Inputs, information will display based on selections to help you in completing the section.

  • Truss Height: The total truss height will display here including overhangs. Refer to the Truss Height display to determine if the section needs to have the piggyback modifier applied.
  • Area: The area in square feet for the seciton will display here.
  • Truss and Gable Count: The number of trusses and gables will display here except when Hip is selected for one or both ends. Truss count will not include girders.
  • Truss Run: If the Area Input method is being used, instead of the Area display, the Truss Run will be calculated and displayed in its place.

Additional Details

  • End 1, End 2: When top row inputs are entered, pricing will reflect common trusses including a truss at each end. The dropdown selections here allow you to make appropriate adjustments for each end. If any of these conditions apply, it is recommended that you select from these dropdowns before using any of the functions below. You can select either "Gable", "Hip", "Valley", "Girder" or "Valley + Girder". When "Hip" is selected on on either end, a dropdown box will appear below (End Slope 1 or End Slope 2). Enter the end slope of the hip end. When "Valley" is selected on either end, a dropdown box will appear below (End Slope 1 or End Slope 2). Enter the underside slope of the valley set. When "Girder" is selected, an input will appear below (End 2 Tie). Enter the girder tie-in (span of trusses being supported). When "Valley + Girder" is selected, enter the underside slope of the valley set (End Slope 1 or End Slope 2) and enter the girder tie-in (span in feet).
  • Modify: Further modifications can be made through the "Modify" dropdown. The method of application changes slightly depending on which Input Method is being used for the specific section.

    Default Method

    If the Default Method is being used for the section (Span and Run inputs), modifications are applied by selecting the modifier and entering the truss run of the specific modifier. Select the modifier that most closely matches the plan specifications. Take the truss run for the trusses affected in the specific area. For example, if you have a 28 x 40 building and half of them are scissor trusses, select the Scissor modifier and enter 20 for the run of trusses that will become scissor trusses. Multiple inputs can be made by making a new selection from the "Modify" dropdown and clicking the "+" button to apply it. Once applied, your modification will display in the Applied Factors list. You can apply mulitple modifiers in the same section.
    1. Scissor: Use this selection for standard scissor trusses with a reasonable spread between top and bottom chord slopes. Enter the truss run affected by this modification in the Run input box.
    2. Max Scissor: This selection can be used when the difference between top and bottom chord slopes are at a minimum. Ceiling slope is near the maximum possible slope. Enter the truss run affected by this modification in the Run input box.
    3. Parallel Chord: This selection is used for Parallel Chord Scissor Trusses. As spans increase, a deeper depth will be assumed. Decreased spacing may help to reduce depth slightly, but typically a truss depth slightly greater than one inch per foot of truss span should be expected. Larger spans and slopes over 7/12 may return a price of $0. Watch the Total section to ensure that the Applied Factor has been applied to the total. Finally, enter the truss run affected by this modification in the Run input box.
    4. Sloped Flat: This selection is used for Sloped Flat trusses. For this area, as with Parallel Chords, it should be expected that the depth will increase as spans increase. Enter the truss run affected by this modification in the Run input box.
    5. Tray Ceiling: This selection is used for trusses with a Tray Ceiling built in. Enter the truss run affected by this modification in the Run input box.
    6. Hip Roof: This selection is used for Hip or Cottage style roof trusses. Include the jacks and corner sets as part of the hip area. Enter the truss run affected by this modification in the Run input box.
    7. Built-In Valley: This selection is used for Built-in Valley trusses. Adjustments will be made to include the increase in joints, webs and pitch breaks. Enter the truss run affected by this modification in the Run input box.
    8. Attic Truss: There can be tremendous variations in Attic truss designs. For this modification, pricing will reflect an attic truss with a maximum of 12' wide room with a 2x8 bottom chord. Enter the truss run affected by this modification in the Run input box.
    9. Mono Truss: This selection is used for Mono trusses. Enter the truss run affected by this modification in the Run input box.
    10. Piggyback: This selection is used for Piggyback trusses. Each section you generate in the "Truss Estimator" will give you an approximate height for the trusses in the specific section. You can refer to this height to see if the trusses will need this modification applied. Enter the truss run affected by this modification in the Run input box.
    11. 2x6 Top Chord: To change pricing to reflect 2x6 Top Chords, use this selection. Enter the truss run affected by this modification in the Run input box.
    12. 2x6 Bottom Chord To change pricing to reflect 2x6 Bottom Chords, use this selection. Enter the truss run affected by this modification in the Run input box.

    Area Input Method

    If the Area Input method is being used for the section, modifications are applied as a percentage of the area square footage of the specific section. Select the modifier that most closely matches the plan specifications. Take the percentage of the area for the trusses affected in the specific area. For example, if you have 1000 square feet entered in the "Area" input in the top row, and 500 square feet is covered by Scissor Trusses, select the modifier and enter 50 for the "%" input. Multiple inputs can be made by making a new selection from the "Modify" dropdown and clicking the "+" button.
    1. Scissor: Use this selection for standard scissor trusses with a reasonable spread between top and bottom chord slopes. Select the % of truss area affected by this modification.
    2. Max Scissor: This selection can be used when the difference between top and bottom chord slopes are at a minimum. Ceiling slope is near the maximum possible slope. Select the % of truss area affected by this modification.
    3. Parallel Chord: This selection is used for Parallel Chord Scissor Trusses. As spans increase, a deeper depth will be assumed. Decreased spacing may help to reduce depth slightly, but typically a truss depth slightly greater than one inch per foot of truss span should be expected. Larger spans and slopes over 7/12 may return a price of $0. Watch the Total section to ensure that the Applied Factor has been applied to the total. Finally, select the % of truss area affected by this modification.
    4. Sloped Flat: This selection is used for Sloped Flat trusses. For this area, as with Parallel Chords, it should be expected that the depth will increase as spans increase. Select the % of truss area affected by this modification.
    5. Tray Ceiling: This selection is used for trusses with a Tray Ceiling built in. Remember to take the percent area for the trusses affected. (The full span of the truss multiplied by the truss run.)
    6. Hip Roof: This selection is used for Hip or Cottage style roof trusses. Include the jacks and corner sets as part of the hip area. Select the % of truss area affected by this modification.
    7. Built-In Valley: This selection is used for Built-in Valley trusses. Adjustments will be make to include the increase in joints, webs and pitch breaks. Select the % of truss area affected by this modification.
    8. Attic Truss: There can be tremendous variations in Attic truss designs. For this modification, pricing will reflect an attic truss with a maximum of 12' wide room with a 2x8 bottom chord.
    9. Mono Truss: This selection is used for Mono trusses. Select the % of truss area affected by this modification.
    10. Piggyback: This selection is used for Piggyback trusses. Each section you generate in the "Truss Estimator" will give you an approximate height for the trusses in the specific section. You can refer to this height to see if the trusses will need this modification applied. Select the % of truss area affected by this modification.
    11. 2x6 Top Chord: To change pricing to reflect 2x6 Top Chords, use this selection. Select the % of truss area affected by this modification.
    12. 2x6 Bottom Chord To change pricing to reflect 2x6 Bottom Chords, use this selection. Select the % of truss area affected by this modification.
  • Add Girder: When a Girder is required within the specific section, for example a girder holding up a dormer, that would not be included in either “End 1” or “End 2” you can add it here. Enter the girder span in linear feet and the tie-in of the trusses the girder is supporting. You can add multiple girders by hitting the "+" button and repeating the process.
  • Add Valley: If you have a Valley Set that is not covered by the "End 1" and "End 2" dropdowns, you can add it in here. There are two ways to input Valley Sets here. If the top chord slope is equal to the underside roof slope, select "Valley Span (ft)" from the dropdown and enter the span in feet in the second box input. (a) Valley Span: If the valley set is behind a girder truss, enter the span of the girder. If you have a gable / valley set overtop and want the valley set to start with a valley truss, take the spacing multiplied by two and add this to the longest valley span. It will be assumed that the first valley will be up the roof slope 1x the spacing distance from the span entered. (b) Valley Area (sq. ft): If the top slope and underside slope are not the same,use this selection from the dropdown. Calculate the area covered by the valley set in sqare feet and enter the value in the second input box.
  • Ladders: This tool will add in gable ladders 4 feet wide. Enter the total horizontal run of the ladders (include full span plus overhangs). Ladders will be priced with 2x4 lumber. You can choose either 2x4 or 2x6 fascia from the dropdown box.

Applied Factors

All modifiers, girders, valleys or ladders added in the "Additional Details" section will be listed in the "Applied Factors" section after the selections are added by clicking the corresponding "+" buttons. If you wish to remove any of these Applied Factors, clicking the "X" button will remove it from the list.


Description

This input box is for your notes on the particular section being priced and will be included in the Notes display when you go to print the quote. When a quote has been saved, notes in this area are helpful to describe the area being quoted and to help make sense of the input selections afterwards. You may want to describe the rooms from the floor plan that are being covered in this area, or which truss components will or will not be included.


Totals

To the right of the screen, the Totals section acts as a dashboard so that you can monitor changes being made to the end price as inputs are filled in or applied. Be aware of expected trends and keep a watch on how changes made affect the resulting totals. You can also adjust your Markup and add Delivery in the input fields of this section. Totals and percentages are also displayed for Lumber, Plates, Labor, Overhead and Delivery.



Print Quote


Overview

The "Print Quote" tab is where you can prepare quotes to print by compiling totals from the "Floor Estimator" and "Truss Estimator", and add other products from "Product Codes".

Quote Items

The "Floor Estimator" total will appear as a Quote Item if the "Floor Estimator" contains a value. "Truss Estimator" total will appear as a separate Quote Item if the "Truss Estimator" contains a value.

  • Hide:If you check the Hide box for a particular Item, it will not display on the printed quote. The dollar total will still be applied to the total price even if it is not displayed. You can use thi feature to hide the item and replace it with your own text information.
  • Edit: If you select the "Edit" button for a particular item, you can edit the Quantity, Item identifier, Span if applicable, and you can add Notes to further describe the listed item. When editing is complete, select "Confirm" to update the item, or "Cancel" to revert back to its previous state.
  • Notes:Text can be added to describe the particular Quote Item. For Truss Packages, exensive text can be added to describe the quote. Text will render line for line as it is entered in the text box here.
  • +Add Quote Item: Click this button to add a line item to the quote. Now you can add additional items such as beams or hangers that you have entered in the "Product Codes" tab. Enter the quantity, then select the Product Type from the dropdown, then in the Description column, select the product from the dropdown.

Print Preferences

  • Quote Options: Certain aspects can be shown or hidden on Print Preview by checking or unchecking the selections. Line total Amount will display by default.
  • Save Quote: This will store the quote in the database so that it can be viewed in "View Quotes."
  • Print Preview: Before sending the quote to a printer, select Print Preview. In this view, click the Print button in the top right to print, or click "Back" to exit.
  • Print Quote / Job Summary: You can view a Job Summary page instead of the completed quote by toggling between Print Quote and Job Summary.


Roof Details

When a roof is quoted, the app will generate a count for Roof Area (plywood and shingles), eave line, raked eave line, ridge line, hip line and valley line in linear feet. These numbers are generated using several assumptions, but will produce a reasonable result. Assumptions for calculations include but are not limited to: matching fascia heights, equal overhangs, valley areas to lay over another section (subtracting areas below the valley sets).



View Quotes


Overview

Saved quotes are stored in the "View Quotes" tab. If material pricing has changed or you need to edit or reprint the quote, you can restore the quote by clicking the Restore Quote button in the far left column. A pop up window will prompt you to confirm the selection. You'll be redirected to the "Project Info" tab, and all selections made for the quote will be restored and ready for modifications.



Company Info


Overview

This is your company information. If you see anything wrong you can edit your company address, phone numbers or email address that you'd want displayed on your quote sheet. In the Estimator section, add your company user names. In this tab you can also upload a company logo to display on your printed quotes.



Job Defaults


Overview

This tab is an important part of customizing SUM+IT to function right for your company. Set design and pricing preferences to be re-used with each new quote. For changes to be stored for future use, scroll to the bottom of the screen and click "Save".


Selected Values

  • Category: Select between Residential and Commercial loading.
  • Snow Load: Select the most common Snow Load for your area. Use the full Ground Snow Load. Saved selections will appear on future quotes in the "Project Info" tab.
  • Rain Load: Select the most common Rain Load for your area. Saved selections will appear on future quotes in the "Project Info" tab.
  • Bearing Width: Enter the default wall width in inches. This will be used for determining girder plies based on bearing requirements.
  • Hip Jack Type: Select the condition that matches your design preferences.
  • Hip Corner Type: Select the condition that matches your design preferences.
  • 2x3 Web Length: Select the condition that matches your design preferences. If you do not have 2x3 in inventory, select "Do not Limit 2x3 Web Length." The correction will be made in the "Cost Variables" tab.
  • Truss Spacing: Select the condition that matches your design preferences.
  • Ladder Run Display: If you build Gable Ladders and want to be able to price them in the "Truss Estimator", select "Show Ladder Run". If you do not sell Ladders, you can hide them from view in "Truss Estimator" here.
  • Heel Height: Select the condition that matches your most common design preference.
  • Overhang: Enter horizontal overhang distance in inches. Select the condition that matches your most common design preference.
  • Truss Markup: Enter your default Markup %. If you use Margin, enter it here as Margin %. If you use Factor of Labor, enter the Factor here.
  • Floor Joist Spacing: Select the Spacing that matches your most common design preference.
  • Taxes: Taxes can be modified to match your local needs. If your province requires provincial and federal tax, you can name and set the rate required for each. If both fields are not required, enter N/A for the tax name.


Cost Variables


Overview

This tab is of top importance for setting up SUM+IT. Labour variables, Lumber costs, plate costs and Markup preferences are all set in "Cost Variables". Once set up properly, this area will be used only to update changes to material costs.


Labour Variables

SUM+IT is constantly improving and adding to the Labour Variables. This is the most difficult part of the process and in most cases, setting up your labour variables will require assistance from a SUM+IT technical advisor.


Markup Method

If you apply either Margin or Markup, you can toggle it here. A third alternative is available to tie profits to plant time. Factor of labor will add a multiple of the labor total as the markup.


Material Cost

  • Lumber Cost: SUM+IT has provisions for one grade of 2x3 and two grades of 2x4, 2x6 and 2x8. You may have a larger lumber inventory. Take your current costs and fit them into these categories the best you can. If you only have one grade of a certain depth, be sure to match up both grades of that lumber depth with the same number.
  • 2x4 Webs: If you do not have 2x3 in your inventory but begin your webs at 2x4 or greater, check the "2x4 Webs" box. You'll notice the 2x3 cost input will gray out but it will contain a value. This will correct the app to ensure that 2x4 will be priced in place of 2x3.
  • Plate Cost: Enter your cost for 20 gauge, 18 gauge and 16 gauge plates. If you do not carry all three gauges of plates, do not leave a field blank. Instead, assign a value similar to the gauges that you carry.

Overhead

If you use Overhead as a part of your pricing method, enter it here.



Product Codes


Overview

In this tab, various products that you sell can be stored in the app for use in the "Floor Estimator" and in the "Print Quote" tab.

  • Adding Items: In each product category, new items can be added by selecting the "+ Add Item"
    • buttons at the top right of each category. Once selected, you can assign a Product Code, product Description, cost, markup and delivery amount for the item. Beam, Joist and Miscellaneous Items should have a price per foot for proper function in the "Floor Estimator". When information is completed, select "Confirm" in the far left column. To undo the entry, click "Cancel." Once confirmed and added, items can be edited with the "Edit" button. When changes are complete, click "Confirm."
  • Item Categories: Product Codes are divided into categories for proper function in "Floor Estimator". Enter products under the appropriate headings. The "Additional Items" category is not used in the "Floor Estimator", but items there can be added to quotes when items are compiled in the "Print Quote" tab.
